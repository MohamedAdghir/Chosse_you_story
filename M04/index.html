El núcleo de las realidades

Realidades Fracturadas es un juego de aventuras narrativas ambientado dentro de una cueva que actúa como núcleo del sistema. Desde este espacio, los jugadores acceden a diferentes aventuras conocidos como realidades fracturadas.

El usuario no vive la historia directamente, sino que observa y decide a través de los personajes.

Cada realidad contiene su propia historia y decisiones. No existe una única aventura, sino múltiples caminos que se activan según el personaje y las elecciones realizadas.

Acceso al sistema

El jugador accede al sistema creando un usuario. Este usuario representa a la persona que explora la cueva y gestiona las decisiones realizadas.

Selección de personaje

Dentro de la cueva, el usuario elige uno de los personajes disponibles. Cada personaje tiene su propio contexto, información y forma de interactuar con las realidades.

Exploración de realidades

Un mismo personaje puede acceder a distintas aventuras. Según el personaje elegido, las opciones, eventos y finales de cada realidad pueden variar.

Un sistema de caminos ramificados

El sistema registra cada decisión y genera nuevas bifurcaciones. No todos los caminos están disponibles para todos los personajes.
